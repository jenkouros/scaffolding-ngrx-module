# Prerequisite

Installed packages:

- @ngrx/effects
- @ngrx/store
- @ngrx/store-devtools

To install use:
`npm install -s @ngrx/store @ngrx/effects @ngrx/store-devtools`

# Installation

`npm install --save-dev '@jenko.uros/ngrx-module'`

# Run schematics

`ng add '@jenko.uros/ngrx-module' your-module-name`
